# Startup LaunchPad

An AI-enhanced management hub for early-stage startups using React, Node.js, and PostgreSQL. This project is being developed by MCA students as part of their 3rd and 4th-semester curriculum.

## Overview

[cite_start]Startup LaunchPad directly addresses the critical pain points for early-stage founders: operational inefficiency, financial mismanagement, and talent acquisition hurdles[cite: 6]. [cite_start]It provides a single, integrated platform that consolidates project management, financial tracking, and a talent marketplace into a seamless experience[cite: 21].

## Core Features

-   [cite_start]**Operations Hub:** An intuitive Kanban board for agile project and task management[cite: 10].
-   [cite_start]**Financial Hub:** A streamlined expense logger and dashboard to visualize burn rate and spending[cite: 12].
-   [cite_start]**Talent Marketplace:** A platform for founders to connect with freelance and student talent[cite: 13].
-   [cite_start]**AI Intelligence Layer:** Proactive features like sentiment analysis, anomaly detection, and talent matching to assist founders[cite: 36].

## Technology Stack

-   **Frontend:** React.js
-   **Backend:** Node.js with Express.js
-   **Database:** PostgreSQL

## Getting Started

### Prerequisites

-   Node.js (v18.x or later)
-   npm / yarn
-   PostgreSQL

### Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/your-username/startup-launchpad.git](https://github.com/your-username/startup-launchpad.git)
    cd startup-launchpad
    ```

2.  **Setup Backend:**
    ```bash
    cd server
    npm install
    cp .env.example .env
    # Update .env with your database credentials
    npm run dev
    ```

3.  **Setup Frontend:**
    ```bash
    cd client
    npm install
    npm start
    ```
