# Goal
To maintain project momentum and clarity through systematic Agile workflows, ensuring timely delivery of the Startup LaunchPad platform milestones.

# Step-by-Step Logic
1. **Backlog Grooming**: Regularly review and prioritize the project backlog based on stakeholder value and technical debt.
2. **Sprint Planning**: Define 2-week sprints with clear objectives and capacity-balanced task assignments.
3. **Kanban Oversight**: Manage the flow of work from "To Do" to "Done," identifying and resolving bottlenecks in real-time.
4. **Gantt Chart Visualization**: Maintain a high-level timeline for major milestones (e.g., MVP Launch, Beta Testing) to track long-term progress.
5. **Retrospectives**: Conduct end-of-sprint reviews to identify process improvements and celebrate wins.

# Technical Constraints
- **Framework**: Scrum or Kanban (Project-dependent).
- **Metric Tracking**: Monitor Velocity (Scrum) or Lead Time (Kanban).
- **Documentation**: All sprint goals and retro action items must be recorded.

# Code Patterns
```markdown
# Sprint [X] Goal: Implement Core Investment Flow
- [ ] Define Investment Data Model (ERD)
- [ ] Implement Backend API for Payments
- [ ] Build Frontend "Invest Now" Component
- [ ] Conduct E2E Testing for Happy Path
```

# Tool Integration
- **Google Search**: Research best practices for agile sizing (e.g., Fibonacci vs. T-shirt sizing).
- **Draw.io**: Create Gantt charts for multi-month project roadmaps.
- **Figma**: Reference design milestones to align with sprint planning.
