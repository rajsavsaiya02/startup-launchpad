# Startup LaunchPad - Server

This directory contains all the backend code for the Startup LaunchPad project. It is a Node.js application using the Express.js framework to build a RESTful API.

## Core Responsibilities

-   User Authentication & Authorization
-   Business Logic for all application features
-   Database interactions with our PostgreSQL instance
-   Serving data to the React frontend

## Getting Started

1.  Navigate to this directory: `cd server`
2.  Install dependencies: `npm install`
3.  Create a `.env` file from the example: `cp .env.example .env`
4.  Update the `.env` file with your database credentials and other secrets.
5.  Start the development server: `npm run dev`

The server will be running on the port specified in your configuration.
