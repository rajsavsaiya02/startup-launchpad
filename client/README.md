# Startup LaunchPad - Client

This directory contains the frontend React application for the Startup LaunchPad project. It was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Core Responsibilities

-   Providing the entire user interface (UI) and user experience (UX).
-   Making API calls to the backend server to fetch and display data.
-   Managing frontend state and user interactions.

## Getting Started

1.  Navigate to this directory: `cd client`
2.  Install dependencies: `npm install`
3.  Start the development server: `npm start`

Open [http://localhost:3000](http://localhost:3000) to view it in the browser. The page will reload if you make edits.
